This `RegExp3.txt` is used to convert some expressions of Jekyll kramdown to markdown in NppMardownPanel:

```cs
	// Comment: {{ site.baseurl }} -> ..
\{\{ site\.baseurl \}\}
..
	// ...{:style -> ...{style
(\S\{):
$1
```
